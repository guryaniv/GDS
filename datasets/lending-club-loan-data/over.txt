These files contain complete loan data for all loans issued through the
2007-2015, including the current loan status (Current, Late, Fully Paid, etc.)
and latest payment information. The file containing loan data through the
"present" contains complete loan data for all loans issued through the
previous completed calendar quarter. Additional features include credit
scores, number of finance inquiries, address including zip codes, and state,
and collections among others. The file is a matrix of about 890 thousand
observations and 75 variables. A data dictionary is provided in a separate
file. k
